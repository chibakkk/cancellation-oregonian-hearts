# Changelog

このプロジェクトの主な変更履歴をまとめます。  
バージョン番号は [Semantic Versioning](https://semver.org/lang/ja/) を参考にしています。

## [0.1.0] - 2026-07-19

### 追加

- 友人との公開テスト向けに Render へデプロイ。
- 4-10人対応のリアルタイム対戦テーブルを実装。
- ルーム作成、ルーム参加、4桁パスワードによる入室を実装。
- ブラウザを閉じた後のセッション復帰導線を追加。
- Redis によるルーム状態、セッショントークン、部屋別ロック、レート制限の保持に対応。
- トリック終了時のプレビュー、勝者表示、カード集合アニメーションを追加。
- ラウンド終了UIとゲーム終了UIを追加。
- トップページを参加中心のUIに変更し、ルーム作成ページを `/create-room` として分離。
- アプリ内ルールページ `/rules` を追加。
- Playwright E2E を追加し、4人卓/10人卓、リロード復帰、タブ復帰、公開URLスモークテストを確認できるようにした。
- Docker Compose スモークテストと公開URL E2Eスクリプトを追加。
- GitHub Actions CI を追加。

### 変更

- トップページの見た目をカードテーブル風に刷新。
- タイトル表示を英語ロゴ寄りに変更し、ブラウザタイトルもアプリ名に変更。
- 参加できないカードの見え方、マストフォロー時のUX、出せない理由表示を改善。
- トリック中のキャンセルカード、リードスート外カード、勝利カードの見え方を改善。
- 無失点ボーナスの余りを次ラウンドへ持ち越すように変更。
- 初期スコアを0点から100点に変更。
- ルール説明に、配り残りカードと無失点ボーナスの割り振りを追記。
- サーバー側の主要エラーメッセージを日本語として読めるように整理。
- README を日本語化し、遊び方、ルール概要、開発/運用手順を整理。

### 注意

- 現在は公開テスト版です。
- Render 無料枠では、アクセスがない時間が続くとサーバーがスリープする場合があります。
- スマホ表示は今後の改善対象です。
- ルール説明は今後も実プレイのフィードバックに合わせて拡充予定です。
