# client

Cancellation Oregonian Hearts のブラウザクライアントです。

## 起動

```powershell
npm install
npm run dev
```

デフォルトでは `VITE_SERVER_URL` 未設定時に `http://localhost:3001` のサーバーへ接続します。

## 主なコマンド

```powershell
npm run build
npm run test:e2e
npm run test:e2e:compose
```

詳しい開発/運用手順はリポジトリルートの `README.md` を参照してください。
